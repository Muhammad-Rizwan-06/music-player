# Music Player

A lightweight and user-friendly frontend music player application. Play, manage, and organize your local music files directly in your browser.  

---

## Features

- Play, pause, stop, and skip tracks  
- Support for popular audio formats (MP3, WAV, AAC, etc.)  
- Shuffle and repeat modes  
- Display song metadata (title, artist, album, duration)  
- Volume control and mute/unmute  
- Responsive and intuitive UI  

---

## Technologies Used

- **HTML** for structure  
- **CSS / Tailwind / Bootstrap** for styling  
- **JavaScript** for functionality and audio control  
- **Web Audio API** for handling audio playback
